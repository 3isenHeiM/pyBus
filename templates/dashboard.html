Welcome

Welcome to the PyBus web interface! Here you can navigate through various pages which will allow you to administer the pyBus software and other components. You will also be able to upload updates to the software and view logs.

Have fun!

Background

I began this project after getting a BMW 320d and wishing I could play music from a memory stick. Since then, the project has evolved to include this web-interface, and add features to the car, such as locking/unlocking automatically and putting up the windows!
